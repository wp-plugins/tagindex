Create index of tags

== Description ==

Tagindex is a tags navigation plugin for wordpress. While categories is similar to table of contents of a book, tags is similar to index.

== Installation ==

1. Upload the folder `tagindex` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a page, use [tagindex] shortcode in the page's content to show the tags index.
